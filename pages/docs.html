Documentation

Connection

Server URL: https://seo.ezbizservices.com/mcp

Auth: Pass your API key as X-API-Key header or api_key query parameter.

Tools

keyword_research
Research keyword opportunities for a business — search volume indicators, difficulty estimates, related terms, and content suggestions.

Parameters:
- seed_keyword (string): Primary keyword or topic to research
- industry (string, optional): Business industry or niche
- location (string, optional): Target geographic location

analyze_serp
Analyze search engine results for a query — top ranking pages, content patterns, SERP features, and ranking opportunity assessment.

Parameters:
- query (string): Search query to analyze
- num_results (number, optional): Number of results to analyze (max 10)

check_backlinks
Analyze a website's backlink profile — referring domains, anchor text patterns, link quality indicators, and link building opportunities.

Parameters:
- url (string): Website URL to analyze
- competitor_urls (string, optional): Comma-separated competitor URLs for comparison

optimize_content
Analyze and optimize content for SEO — keyword density, readability, structure, meta tags, and actionable improvement suggestions.

Parameters:
- url (string): URL of the page to optimize
- target_keyword (string): Primary keyword to optimize for

API Endpoints

  • POST /mcp — MCP protocol endpoint
  • POST /api/keys/signup — Create free API key (body: {name, email})
  • GET /api/keys/usage?key=YOUR_KEY — Check usage
  • GET /api/pricing — View pricing tiers
  • GET /health — Health check
